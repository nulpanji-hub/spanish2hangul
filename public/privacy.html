Política de Privacidad

세종의 후예 (Descendants of Sejong)
Última actualización: 29 de marzo de 2026

Esta Política de Privacidad describe cómo la aplicación "세종의 후예" (en adelante, "la Aplicación") recopila, usa y protege la información cuando utilizas nuestros servicios.

🔒Compromiso de Privacidad: La Aplicación está diseñada para respetar tu privacidad. No recopilamos información personal identificable y no compartimos tus datos con terceros con fines comerciales.

1. Información que Recopilamos

1.1 Datos que SÍ procesamos

  • Texto ingresado: Las frases en español que escribes para convertir a Hangul se procesan a través de nuestra IA (Google Gemini API).
  • Imágenes capturadas (OCR): Cuando usas la función de escaneo, las imágenes se procesan localmente en tu dispositivo y a través de la API para extraer texto. Las imágenes no se almacenan en nuestros servidores.
  • Grabaciones de voz: Cuando usas la entrada por voz, el audio se procesa temporalmente para convertirlo en texto. El audio no se almacena permanentemente.
  • Registros de uso de API: Información técnica sobre las solicitudes a Google Gemini API (sin contenido personal).

1.2 Datos que NO recopilamos

  • Nombre, correo electrónico o número de teléfono
  • Información de ubicación geográfica
  • Información de pago (la compra se gestiona exclusivamente a través de Google Play Store)
  • Datos de cuenta de usuario (la Aplicación no requiere registro)
  • Historial de navegación o comportamiento fuera de la Aplicación

2. Cómo Usamos la Información

La información procesada se utiliza únicamente para:

  • Conversión de texto: Transformar tu entrada en español a la pronunciación en Hangul usando inteligencia artificial.
  • Generación de ejemplos personalizados: Crear frases de ejemplo basadas en noticias actuales de BTS y K-pop mediante Google Gemini API.
  • Mejora del servicio: Optimizar la precisión de las conversiones fonéticas.

⚠️Importante: Tus datos de entrada (texto, imágenes, voz) no se almacenan de forma permanente en nuestros servidores ni se utilizan para entrenar modelos de IA sin tu consentimiento.

3. Compartir Información con Terceros

3.1 Google Gemini API

La Aplicación utiliza Google Gemini API para procesar las conversiones de texto. Esto significa que:

  • El texto que ingresas se envía a los servidores de Google para su procesamiento mediante IA.
  • Google puede procesar estos datos según su propia Política de Privacidad.
  • No compartimos tus datos con Google con fines publicitarios o de marketing.

3.2 No vendemos tus datos

No vendemos, alquilamos ni compartimos tu información personal con terceros para fines comerciales o publicitarios.

4. Almacenamiento y Seguridad de Datos

4.1 Datos locales

La mayoría del procesamiento ocurre localmente en tu dispositivo. Las imágenes capturadas con OCR y las grabaciones de voz se procesan de forma temporal y no se almacenan permanentemente.

4.2 Datos en tránsito

Las comunicaciones entre la Aplicación y Google Gemini API están cifradas mediante HTTPS para proteger tus datos durante la transmisión.

4.3 Período de retención

Los datos procesados a través de la API se eliminan inmediatamente después de completar la conversión. No mantenemos un historial de tus conversiones.

5. Permisos de la Aplicación

La Aplicación solicita los siguientes permisos para funcionar correctamente:

  • 📷Cámara: Para escanear texto de imágenes (función OCR). Solo se usa cuando tú activas esta función.
  • 🎤Micrófono: Para la entrada de voz. Solo se usa cuando tú activas esta función.
  • 🌐Internet: Para conectarse a Google Gemini API y generar ejemplos personalizados.

Puedes revocar estos permisos en cualquier momento desde la configuración de tu dispositivo.

6. Derechos del Usuario

Como usuario de la Aplicación, tienes derecho a:

  • Acceso: Solicitar información sobre qué datos procesamos (aunque no almacenamos datos personales).
  • Eliminación: Solicitar la eliminación de cualquier dato que podamos tener (si aplicable).
  • Portabilidad: Solicitar una copia de tus datos en formato legible (si aplicable).
  • Retirar consentimiento: Dejar de usar la Aplicación en cualquier momento.

7. Privacidad de Menores

La Aplicación está diseñada para usuarios de todas las edades, incluidos menores de 13 años. No recopilamos intencionalmente información personal de menores. Si descubres que un menor ha proporcionado información personal sin el consentimiento de los padres, contáctanos inmediatamente.

8. Cookies y Tecnologías de Seguimiento

La Aplicación NO utiliza cookies ni tecnologías de seguimiento para rastrear tu comportamiento.

9. Cambios a Esta Política de Privacidad

Podemos actualizar esta Política de Privacidad ocasionalmente. Te notificaremos de cualquier cambio importante mediante:

  • Una actualización de la aplicación con la nueva política.
  • Una notificación dentro de la Aplicación.
  • Actualizando la fecha de "Última actualización" en la parte superior de esta página.

Te recomendamos revisar esta política periódicamente.

10. Legislación Aplicable

Esta Política de Privacidad se rige por las leyes de protección de datos aplicables, incluyendo:

  • GDPR (Reglamento General de Protección de Datos de la Unión Europea) para usuarios en Europa.
  • Ley de Protección de Datos Personales de Corea del Sur (개인정보 보호법).
  • Leyes locales de protección de datos en países de habla hispana donde se usa la Aplicación.
